# Heard

**You sleep. It replies.**

Heard answers every review across your stores, in your brand voice — and only wakes you for the ones that matter.

---

## What it does

Multi-platform sellers burn time every morning answering reviews across multiple dashboards. Heard runs the overnight shift:

1. Pulls new reviews from every connected store
2. Classifies each: praise / question / complaint / urgent
3. Drafts an on-brand reply, applying your rules (e.g. "never promise refunds publicly")
4. Auto-posts the safe ones
5. Escalates the risky ones to you — with the order already pulled up

## Platforms

- **Judge.me** (Shopify review app) — webhook-driven, real-time
- **Google Business Profile** — poll-driven every 2 hours
- Shopify Admin API for order context on complaint reviews

## Stack

Next.js · Vercel · Supabase · Python ADK · Gemini 2.0 Flash · Google Cloud Run · Cloud Scheduler
